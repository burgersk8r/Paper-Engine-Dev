==  SONGS  ==
Put your songs here, it should look like this:

assets/music/songs/your-song-name-here/
---- Inst.ogg
---- Voices.ogg

Alternatively, it can also look like this if you have separate voice files:

assets/music/songs/your-song-name-here/
---- Inst.ogg
---- Voices-Opponent.ogg
---- Voices-Player.ogg
